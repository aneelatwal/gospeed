# gospeed

### Run this in the project root after cloning the repo:

```
npm install
```

### To compile and run the executable, run this command:
```
make build
make run
```

### Install on Proxmox as LXC:
```
bash -c "$(curl -fsSL https://raw.githubusercontent.com/aneelatwal/gospeed/main/install-gospeed.sh)"
```
